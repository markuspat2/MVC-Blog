Mark Patterson Photography

Challenge

Create a photography portfolio using HTML 5 and CSS3 that highlights the photography above everything else. I wanted this site to be simple and responsive with full page imagery.

Solution

I used HTML 5 for the structure of the page and CSS3 to drive the animations. I used percentages for all of my elements so that they would resize easily to fit on different sized devices.
